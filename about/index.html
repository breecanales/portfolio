Required items for this project:

  1. Content map: I generated it with the Mind Map tool (see below)
  2. Wireframe: I generated this using the online tool www.wireframe.cc (see below)
  3. HTML 5: the structure uses HTML 5 tags such as nav, section and footer
  4. CSS for layout: the site uses basic CSS for all layouts
  5. Top four sections are: Home, About, Samples and Contact
  6. I created the favicon for the site using Photoshop
  7. Glyphs: see sample #4
  8. Video element: see sample #2
  9. Audio element: see sample #6
  10. Slideshow: see homepage (used yii2sly)
  11. Use a web font: every h1 tag uses a web font from Google (also see example #1)
  12. HTML 5 canvas element: see example #3 (drew a traffic cone)
  13. HTML 5 form validation: see example #7
  14. JSON implementation: see example #5 (schools I've attended overlaid on a Google Map)
  15. Server-side persistent storage: all the samples are stored in a SQLite database
  16. Data visualization: used Google services to show three gauges (see example #9)
  17. Asynchronous updates: see example #10
  18. Client-side scripting: used highlight.js to format all code examples; also see example #8
  19. Server-side scripting: this entire application is built with the PHP-driven Yii2 framework
